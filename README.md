## Getting Started

### Prerequisites

- Node.js 14+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

### Build

```bash
# Create production build
npm run build
```

### Environment Variables

Create a `.env` file in the client directory:

```env
VITE_SERVER_URL=http://localhost:3000
```
